Cascading Style Sheet

CSS는 Cascading Style Sheets의 약자로, 웹 페이지의 스타일(디자인)을 지정하는 데 사용되는 언어입니다. HTML이 웹 페이지의 구조(콘텐츠)를 정의한다면, CSS는 그 구조를 어떻게 보이게 할지(색상, 크기, 레이아웃 등) 를 정의합니다.

  • Color and Background
  • Text
  • Font
  • Document
  • Box model
  • Visual format and effect
  • List
  • Table
  • User Interface
